    # Stepper 步进器

    QR
    扫二维码查看演示效果

    用作增加或者减少当前数值。

    # 规则

    当想要对数值进行小幅度调整时，可以使用 Stepper，eg：将年化收益从 4.00% 调整到 4.05%。

    # Code Example

    <template>
  <div>
    <List>
      <ListItem wrap>
        <template slot="extra">
          <Stepper
            :style="{ width: '100%', minWidth: '100px' }"
            :max="10"
            :min="1"
            showNumber
            v-model="value"></Stepper>
        </template>
        Show number value
      </ListItem>
      <ListItem wrap>
        <template slot="extra">
          <Stepper
            :style="{ width: '100%', minWidth: '100px' }"
            :max="10"
            :min="1"
            showNumber
            :defaultValue="3"
            disabled></Stepper>
        </template>
        Disabled
      </ListItem>
    </List>
  </div>
</template>
<script>
import List from '@/components/list'
import ListItem from '@/components/list-item'
import Stepper from '@/components/stepper'
export default {
  components: {
    Stepper,
    List,
    ListItem
  },
  data () {
    return {
      value: 3
    }
  }
}
</script>

    # props

    属性说明类型默认值
    min最小值Number-Infinity
    min最大值NumberInfinity
    v-model当前值 支持双向绑定Number-
    defaultValue初始值Number-
    disabled禁用Booleanfalse
    readOnlyinput 只读Booleanfalse
    showNumber(web only)是否显示数值，默认不显示Booleanfalse
